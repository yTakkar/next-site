Using `withAmp` you can create an AMP only page by supplying your page's component as the parameter to `withAmp`.

```js
// pages/index.js
import { withAmp } from 'next/amp'

export default withAmp(function Index(props) {
  return <p>Welcome to the AMP only Index page!!</p>
})
```
