Using `withAmp` you can also create a hybrid AMP page by supplying your page's component as the first parameter to `withAmp` and a second parameter of `{ hybrid: true }`. To view the AMP version of the page you add `?amp=1` to your URL.

```js
// pages/index.js
import { withAmp, useAmp } from 'next/amp'

export default withAmp(
  function Index(props) {
    const isAmp = useAmp()
    return <p>Welcome to the {isAmp ? 'AMP' : 'normal'} version of the Index page!!</p>
  },
  { hybrid: true }
)
```
