## Setup

For this lesson, we have already built a very basic hacker news clone.
Here is how to get it:

```bash
git clone https://github.com/zeit/next-learn-demo.git
cd next-learn-demo
cd E2-lazy-loading-modules
```

Then you can run the app with:

```bash
npm install
npm run dev
```

Now, visit <http://localhost:3000> and try out the app.
